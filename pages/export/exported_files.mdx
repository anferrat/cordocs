## Exported files

Exported files screen contains all the files that were exported from your survey. To access them, navigate to 'Settings' -> 'Exported Files'. From here, you can share files with third party apps or delete old files.

Tap on tripple dot button next to the file to view available options.

<ImageView>
  <Image src="/images/export/6.9.png" alt="Add" width={300} height={300} />
</ImageView>

Spreadsheet files can be viewed directly from the app. Larger files may not be fully displayed, and editing information within the file is not supported.

To delete all exported files at once, use the "Delete All" button located at the bottom of the screen. This action is irreversible.
