# Supported multimeters

Currently, only one model is supported. It's important to note that Corpad does not assume responsibility for the accuracy of measurements, but rather provides an interface to automatically record readings from the device using Bluetooth technology.

Pokit Pro, crafted by Pokit Innovations, is a versatile Bluetooth multimeter, oscilloscope, and data logger designed for ultimate portability. This CAT III multimeter boasts a compact size with an input impedance of 10M Ohm. Compatible with iOS, Android, Watch OS, and Wear OS, it enhances its functionality through dedicated applications.

**Important Considerations:**

- Compact and Precise: A CAT III multimeter tailored for on-the-go professionals.

- Connectivity: Compatible with iOS, Android, Watch OS, and Wear OS for seamless integration.

- Versatility: Functions as a multimeter, oscilloscope, and data logger for comprehensive data capture.

- Limitations: Not water or dust-resistant; consider the environment of use.

- Connector Note: Banana plug connectors are a separate purchase.

- Capture Mode: Supports On/Off cycle capture with cycle times ranging from 1000 to 20000 ms.
