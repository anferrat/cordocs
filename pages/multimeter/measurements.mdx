# Measurements and Graph

You can access connected multimeter to view currently measured readings. It's available from the main menu or from 'Settings' -> 'Digital multimeter' -> 'Measurements'.

//image

Available modes are displayed at the top part of the screen. Switch te toggle on the multimeter (if available) to view other available modes.

//image

The multimeter ranges available at the bottom part of the screen. Select the appropriate range for your measurement in order to get most accurate reading. When measured value is exeeding the maximum value for selected range, the app wll attempt to select different range and "or" flag will be displayed. Make sure to not exceed maximums that are displayed in the yellow box at the top of the screen.

## Controls

You can save currently dsiplayed value with 'Save' button. The values will be stored on the device and can be accesed by pressing 'History' button. Long press on the value, will copy it into the clipboard and you can also export all readings to .csv file by pressing 'Export all' button. The generated file will be available in 'Exported files' section in Settings. 

Hold button will freeze the last recorded value on the screen, untill resume button is pressed.

## Graph mode - beta

//image


To view readings in a chart, press Graph button. Y-axis represents values (e.g. Voltage or Current), while X-axis represents time. You can adjust max values for the axes in by manually entering them in the fields below the chart.
