# Creating survey items

You can create new test points or rectifiers directly in the 'Map' screen. Long press on an empty space on the map where you wish to create a new item. Then, select the desired type from the options that appear.

After selecting the type of item, you will be re-directed to 'Edit' screen. Latitude and longitude fields will be automatically filled with coordinates that you selected.
