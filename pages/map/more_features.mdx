# Other features

## Filters

You can filter markers on the map based on their properties. Two filters are supported at this time:

- Status filter
- Marker type filter

//Image 5.10

Refresh button re-loads markers from the database and applies selected filters to displayed markers.

## Search

The search field in the map screen enables quick searching for an item on the map using its name. Only currently visible items are displayed in the search results. Filtered out items are not searchable.
