## Generating QR code

To generate QR code for a test point or rectifier, select 'Create label' button on the 'View' screen and then select 'Generate QR code'.

QR code will be generated shortly. In order to share the code, press 'Export QR code' and an image with the code will be created and placed in 'Exported files' folder.
